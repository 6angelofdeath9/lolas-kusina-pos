# Lola's Kusina Restaurant Order System

A Windows desktop application for restaurant ordering with GUI.

## Features
- Filipino-themed user interface
- Complete menu management
- Order processing with quantities
- Discount system (Student, Senior, PWD)
- Receipt generation
- Payment processing

## Technologies
- C++
- Windows API
- Visual Studio

## How to Run
1. Open `LolasKusinaGUI.sln` in Visual Studio
2. Build the project (Ctrl+Shift+B)
3. Run (F5)
